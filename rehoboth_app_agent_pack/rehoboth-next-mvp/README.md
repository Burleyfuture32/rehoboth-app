# Rehoboth Lending MVP

Next.js + TypeScript + Tailwind + Prisma + SQLite starter for the Rehoboth lending app.

## Current phase

- Dashboard
- Lead intake

## Local run

```bash
npm install
npm run db:push
npm run db:seed
npm run dev
```

If PowerShell blocks `npm`, use `npm.cmd` instead.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Prisma
- SQLite

## Notes

- Single-tenant first
- Local SQLite demo data only
- Beginner-friendly UI first
- Pipeline board, borrower workspace, tasks, and document flow come next
