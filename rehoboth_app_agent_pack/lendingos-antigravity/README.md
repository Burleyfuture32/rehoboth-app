# LendingOS Antigravity Prompt Pack

This folder contains the prompt system for building a clean, beginner-friendly CRM for CRE and residential lending using a multi-agent setup.

## Files

- `01-master-agent-prompt.txt`
  Single strong technical cofounder agent prompt

- `02-strict-mode-addon.txt`
  Add-on to make the agent push back harder

- `03-multi-agent-system.txt`
  Multi-agent system prompt

- `04-single-master-agent-version.txt`
  Compact version for one manager agent

- `05-antigravity-build-crew.txt`
  Antigravity-style named agents, handoffs, packets, and gates

- `06-task-packet-template.txt`
  Founder task packet template

- `07-handoff-block-template.txt`
  Agent-to-agent handoff block

- `08-build-packet-template.txt`
  Builder-focused implementation packet

- `09-qa-packet-template.txt`
  QA-focused validation packet

- `10-mvp-build-order-starter.txt`
  Starter packet for generating the MVP build order

- `11-enforcement-mode-addon.txt`
  Extra guardrails to keep the system disciplined

## Suggested use order

1. `05-antigravity-build-crew.txt`
2. `11-enforcement-mode-addon.txt`
3. `10-mvp-build-order-starter.txt`

## Recommended workflow

Paste the Antigravity build crew prompt first, then append enforcement mode, then send a task packet for the feature or planning work you want done.
