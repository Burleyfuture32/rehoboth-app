# Rehoboth Lending MVP

Investor-demo-ready first slice for the Rehoboth lending app.

## Included in this slice

- dashboard
- lead intake
- pipeline board
- borrower and deal workspace
- task center
- document request and upload view

## Local run

```bash
npm install
npm run dev
```

If PowerShell blocks `npm`, use:

```bash
npm.cmd install
npm.cmd run dev
```

## Notes

- Single-tenant only
- Web-first only
- Seeded local data only
- No auth, notifications, integrations, or non-demo extras

See [docs/first-slice.md](./docs/first-slice.md) for the architecture summary, route map, first tables, endpoints, screens, and implementation order.
