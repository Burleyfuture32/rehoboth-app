# Start Here

This zip includes two setups:

## 1) `lendingos-antigravity/`
The full 11-file system with the build crew, strict mode, task packets, QA packets, and the MVP starter.

## 2) `agents/`
A simpler 8-agent folder if you want to work one agent at a time.

## Fastest way to use from a phone
1. Open `lendingos-antigravity/05-antigravity-build-crew.txt`
2. Paste that into Codex as the system prompt
3. Then paste `lendingos-antigravity/11-enforcement-mode-addon.txt`
4. Then paste `lendingos-antigravity/10-mvp-build-order-starter.txt`
5. Tell Codex to generate the first investor-demo-ready slice of the app

## Best first command to paste after loading the prompts
Create the first investor-demo-ready MVP slice for the Rehoboth lending app.
Focus only on:
- dashboard
- lead intake
- pipeline board
- borrower/deal workspace
- task center
- document request/upload view

Requirements:
- web-first
- calm beginner-friendly UI
- CRE and residential both supported
- single-tenant first
- modular monolith
- no overengineering
- include realistic seed data
- include a clickable demo flow I can show investors
- defer anything not needed for the demo
- return:
  1. architecture summary
  2. route map
  3. first tables
  4. first endpoints
  5. first screens
  6. implementation order
  7. code for the first slice

## Notes
- Use the full pack when you want stricter control and handoffs.
- Use the `agents/` folder when you want to mix and match.
