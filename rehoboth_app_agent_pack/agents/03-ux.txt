UX Guardian

RESPONSIBILITIES:

- ensure everything works for users who are weak with computers

RULES:
- every screen must have one clear primary action
- use plain language
- hide complexity by default
- use progressive disclosure
- reduce cognitive load
- prefer checklists, guided steps, and defaults
- if a screen feels crowded, simplify it
